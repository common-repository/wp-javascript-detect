== Description ==

The WP Javascript Detect plugin is a very simple plugin that provides a way to let website visitors know your website requires the usage of Javascript. There is no special setup required in order to use this plugin, simply install and activate. Once activated the plugin will detect whether or not the visitor has enabled their Javascript, if Javascript is disabled the plugin will display a warning message at the top of the screen.
